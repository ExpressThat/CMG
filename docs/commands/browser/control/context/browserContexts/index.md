# `browser control context browserContexts`

Isolated browser context commands.

```powershell
cmg browser control context browserContexts [command] [options]
```

## Subcommands

- [`new`](new.md): Create and activate a fresh browser context.
- [`newContext`](newContext.md): Create and activate a fresh browser context.
- [`use`](use.md): Activate a browser context by id or target id.
- [`useContext`](useContext.md): Activate a browser context by id or target id.
- [`list`](list.md): List browser contexts.
- [`listContexts`](listContexts.md): List browser contexts.
- [`close`](close.md): Close a browser context by id or target id.
- [`closeContext`](closeContext.md): Close a browser context by id or target id.
