# `browser control input clipboard`

Page-side clipboard shim commands.

```powershell
cmg browser control input clipboard [command] [options]
```

## Subcommands

- [`set`](set.md): Set clipboard shim text.
- [`setClipboard`](setClipboard.md): Set clipboard shim text.
- [`writeClipboard`](writeClipboard.md): Set clipboard shim text.
- [`read`](read.md): Read clipboard shim text.
- [`readClipboard`](readClipboard.md): Read clipboard shim text.
- [`clear`](clear.md): Clear clipboard shim text.
- [`clearClipboard`](clearClipboard.md): Clear clipboard shim text.
