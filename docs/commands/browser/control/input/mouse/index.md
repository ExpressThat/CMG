# `browser control input mouse`

Low-level mouse movement and button commands.

```powershell
cmg browser control input mouse [command] [options]
```

## Subcommands

- [`move`](move.md): Move the mouse.
- [`down`](down.md): Press the mouse button.
- [`up`](up.md): Release the mouse button.

## Target Forms

Each command accepts one target alias such as `center`, `top`, `bottom`, `left`, `right`, `topLeft`, `topRight`, `bottomLeft`, or `bottomRight`; coordinate options `--x` and `--y`; or selector-edge options `--selector`, `--edge`, and `--inset`.
