# `browser control network webSocket`

WebSocket route and wait command group.

```powershell
cmg browser control network webSocket [command] [options]
```

## Subcommands

- [`route`](route.md): Install a WebSocket route.
- [`clearRoutes`](clearRoutes.md): Clear WebSocket routes.
- [`wait`](wait.md): Wait for a matching WebSocket.
- [`waitMessage`](waitMessage.md): Wait for a matching WebSocket message.
