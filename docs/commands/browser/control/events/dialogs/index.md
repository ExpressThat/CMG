# `browser control events dialogs`

Browser dialog capture, behavior, and wait commands.

```powershell
cmg browser control events dialogs [command] [options]
```

## Subcommands

- [`capture`](capture.md): Install dialog capture with default accept behavior.
- [`captureDialogs`](captureDialogs.md): Install dialog capture with default accept behavior.
- [`behavior`](behavior.md): Set automated browser dialog behavior.
- [`setDialogBehavior`](setDialogBehavior.md): Set automated browser dialog behavior.
- [`onDialog`](onDialog.md): Set automated browser dialog behavior.
- [`handleDialog`](handleDialog.md): Set automated browser dialog behavior.
- [`dialogBehavior`](dialogBehavior.md): Set automated browser dialog behavior.
- [`wait`](wait.md): Wait for a matching browser dialog.
- [`waitForDialog`](waitForDialog.md): Wait for a matching browser dialog.
