# E2E Script Action Coverage

This file is the script action behavior coverage manifest for the E2E suite. `ScriptActionCoverageE2eTests` checks every backticked action name in `docs/scripting/actions.md` has an owner row here and that every owner file exists.

This is a coverage ownership map, not a claim that every argument, option, and failure permutation is complete. Add focused rows when an action grows behavior that needs its own E2E file.
